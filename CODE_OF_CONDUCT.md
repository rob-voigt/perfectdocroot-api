# Code of Conduct

PerfectDocRoot aims to build a welcoming and collaborative community.

We expect contributors to:

• be respectful and constructive  
• focus discussions on the technology and ideas  
• help create a positive developer environment  

Unacceptable behavior includes:

• harassment
• personal attacks
• discriminatory language

Project maintainers may remove or reject contributions that violate this code of conduct.

By participating in this project, you agree to follow these guidelines.
